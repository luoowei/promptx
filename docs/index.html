PromptX

Your AI-powered terminal assistant. Ask questions, generate commands, debug errors — all from the command line.

$ px ask "find all large files over 100MB, sort by size"

Here's the command to find and sort large files:
$ find . -type f -size +100M -exec ls -lh {} \; 2>/dev/null | sort -k5 -hr

This searches recursively from current directory, lists files > 100MB, and sorts by size descending.

Install in Seconds

# macOS / Linux
curl -fsSL https://raw.githubusercontent.com/luoowei/promptx/main/install.sh | bash
# Homebrew
brew install luoowei/tap/promptx
# Go install
go install github.com/luoowei/promptx/cmd/px@latest
# Windows (PowerShell)
iwr https://raw.githubusercontent.com/luoowei/promptx/main/install.ps1 | iex

Then set your API key and start:
export OPENAI_API_KEY=sk-... px ask "hello"
